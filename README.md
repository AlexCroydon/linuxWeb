## How to use

Go to [linuxWeb](https://617d5c82f9d62b0007efa555--serene-bhabha-879d38.netlify.app/)  

Build it yourself by running the following command:
```
git clone https://github.com/Manthee1/linuxWeb.git
cd linuxWeb
yarn install
yarn build

```
